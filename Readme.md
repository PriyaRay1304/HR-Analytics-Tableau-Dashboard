HR Analytics Dashboard – Tableau

Overview
This project is an HR Analytics dashboard built using Tableau to analyze employee attrition, workforce demographics, and job satisfaction.

The dashboard helps HR teams understand:
- Overall attrition rate
- Attrition by gender, department, education, and age group
- Job satisfaction distribution across roles
- Workforce age structure

Tools Used
- Tableau Desktop
- Microsoft Excel
- GitHub

Key Metrics
- Total Employees: 1,470
- Attrition Count: 237
- Attrition Rate: 16.12%
- Active Employees: 1,233
- Average Age: 37

Dashboard Preview

Insights
- Highest attrition observed in the 25–34 age group
- R&D and Sales show higher attrition compared to HR
- Male attrition count is higher than female
- Life Sciences and Medical backgrounds contribute the most to attrition
